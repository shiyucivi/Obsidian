### 1 函数返回引用
```rust
fn first_or(string: &Vec<String>, default: &String) -> &String {
    if string.len() > 0 {
		return &string[0]
    } else {
	    return default
    }
}
fn return_string() -> &String {
    let s = String::from("hello");
    &s
}
```
这两个函数都无法编译通过，第一个函数返回了一个引用，但不知道是哪个参数的引用。第二个函数创建了函数体内变量s的引用，但变量s在作用域结束时会被释放，&s变成了悬垂指针。
