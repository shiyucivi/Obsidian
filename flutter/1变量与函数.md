## 1 变量声明
dart是静态类型的语言，变量类型被确定后不能修改。
final和const用于声明常量，const在声明时必须确定值，final可以在运行时才确定值。
```dart
//主函数, 执行的入口
void main(List<String> args) {
	var age = 24;  //变量声明，age被推断为int
	const pi = 3.1415926; //常量声明
	finial time = DateTime.now()
	
	//字符串类型
	String name = "Tome";
	print("Hello, $name, now time is ${DateTime.now()}");  //模板字符串
	
	//数字类型
	int count = 0;  //整型
	num money = 1.5;  //可整型可小数
	double percent = 0.4;  //小数
	count = money.toInt();  //小数转整型
	percent = (count/money).toDouble(); //小数转整型
	
	//布尔类型
	bool trueValue = true;
	
	//列表(数组)
	List studentsList = ["Tom", "Jerry"];
	studentsList.length; //列表长度
	studentsList.last; //最后一个元素
	studentsList.first; //第一个元素
	studentsList.isEmpty; //是否为空列表，布尔值
	//数组方法
	studentsList.add("Harry"); //数组尾部添加元素
	studentsList.addAll(["Jenify", "Snow"]); //尾部添加多个元素
	studentsList.removeLast(); //删除最后一个元素
	studentsList.remove("Jenify"); //删除匹配的第一个元素
	studentsList.removeRange(1, 3); //从第1个元素开始删除，到第4个前为止
	//列表循环
	studentsList.forEach((item){ print(item) });
	//如果都为true，返回false，否则返回true
	studentsList.evety((item){ return true });
	//筛选出返回true的元素
	studentsList.where((item){ if(item == "Tome") return true });
	
	//字典Map
	Map languages = {
		"English": "英文",
		"Chinese": "中文"
	};
	print(Map["Chinses"]); //读取
	Map["Spinish"] = "西班牙文"; //写入或更新
	//字典方法
	//字典遍历
	languages.forEach((key, value){ print("$key is $value") })
	languages.addAll({"France": "法文"}); //字典添加键值对
	languages.containsKey("France");  //是否包含France key
	languages.remove("France"); //移除一个键值对
	languages.clear(); //清空
	
	//dynamic动态类型 编译时不会检查类型
	dynamic freeWord = "free";
	freeWord = true;
}
```

### 2 可空类型
只有可空类型和动态类型可以在初始化时不赋值，此时默认为null。否则必须赋值。
```dart
String? userName = null; //变量可以为null或字符串
userName?.subString(", hello"); //非空判断, 为null时跳过
userName!.length; //断言为非空
username??"Guest"; //空合并，相当于js的||
```
### 3 流程控制
dart是不存在隐式转换，if的判断条件必须为布尔值或者判断表达式
```dart
void main(List<String> args) {
	int score = 61;
	String s;
	//if判断
	if (score > 60) {
		s = "及格";
	} else {
		s = "不及格"
	}
	// switch判断
	switch(String) {
		case "及格":
			print("恭喜");
		case "不及格":
			print("抱歉");
		default:
			print("")	
	}
	//while循环，可以用continue跳过本次循环或break跳出整个循环
	int count = 1
	while(count < 60) {
		count++
	}
	//for循环，也可以用contine和break
	List nums = [1, 2, 3, 4, 5]
	for (var i = 0; i < nums.length; i+=1) {
		if (i % 2 == 0) {
			continue
		}
	}
}
```
### 4 函数
函数声明可以声明返回值类型，也可以声明无返回值，不声明时推断
```dart
//固定参数与可选参数
String combine(String a, [String? b, String? c]) {
	return a + (b??"") + (c??"");
}
combine("Hello", "Tom");
//按参数名传参，此时可以不按顺序传参
String getInfo(String userName, {int? age, String? sex}) {
	if (sex != null) {
		if (sex == "男") {
			print("你好,$userName先生")
		} else {
			print("你好,$userName女士")
		}
	}
}
getInfo('Tom', sex: "男", age: 24);

//匿名函数
Function test = () { print("测试") };
void onTest (Function callback) { callback(); }

//箭头函数
int add (int a, int b) => a + b 
Function subtract = (int a, int b ) => a - b
```
