## 1 属性、方法与构造函数

默认属性和方法都为公有属性。
```dart
class User {
	String name = 'guest';
	int age = 18;
	String _id;
	//构造函数
	User({String name, int age}) {
		this.name = name;
		this.age = age
	}
	void update({String? name, int? age}) {
		if (name != null ) this.name = name;
		if (init != null) this.age = age;
	}
	
}
void main(List<String> args) {
	User user1 = new User(name: "Tom", age: 21);
	user1.update(age: 24);
}
```
构造函数可以用语法糖写法快速初始化参数，==必传参数推荐使用required关键字声明==
```dart
class User {
	String name;
	int age;
	Person({required this.name, required this.age});
}
User user1 = new User(name: "Tom", age: 21)
```
class的属性如果在定义时为空，那么在构造函数中必须进行初始化。
私有属性和私有方法以_开头。
静态属性和静态方法以static关键字声明，必须定义时就初始化。
==new关键字可以省略==
## 2 继承
子类的构造方法中需要调用super函数并传入参数
```dart
class VipUser extends User {
	String phone;
	VipUser({required String name, required int age, required this.phone}): 
		super(name: name, age: age)
}
Vipuser user2 = new VipUser(name = "Sunny", age = "36, phone = "8697267");
```
或者使用super语法糖：
```dart
class VipUser extends User {
	String phone;
	VipUser({required super.name, required super.age, required this.phone});
}
```
使用@override语法糖可以覆盖父类的方法：
```dart
class VipUser extends User {
	String phone;
	VipUser({required super.name, required super.age, required this.phone});
	@override
	update({String? name, int? age, String? phone}) {
		if (name != null ) this.name = name;
		if (init != null) this.age = age;
		if (phone != null) this.phone = phone;
	}
}
```
## 3 抽象类
使用abstract定义抽象类。抽象类中不能有具体赋值或实现。实现具体方法时要@override标注。
```dart
abstract class PayWay {
	String payUrl;
	void getPayUrl();
}
class AliPay implements PayWay {
	String payUrl;
	@override
	void getPayUrl() {
		this.payUrl = 'https://ali.cn';
		return this.payUrl;
	}
}
```
