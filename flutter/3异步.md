dart才用单线程+事件循环机制。执行顺序为：
同步代码 -> 微任务队列 -> 事件队列
## 1 Feature
Feature代表一个异步操作。存在uncompleted（等待）、completed（完成）、error（失败）三种状态。
```dart
Feature f = Future(() {
	return "Hello";
})
//执行成功进入then，抛出异常进入catchError()
f.then((value) {
	print(value)
})
.catchError((error){})
```
## 2 async和await
```dart
void test() async {
	await Feature()
}
```
