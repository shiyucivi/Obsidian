#### 4.1 软件安装
ubuntu使用apt作为软件包管理器，centos使用yum，二者子命令相同
```shell
apt install wegt -y #安装wegt
apt remove wegt #卸载wegt
```
#### 4.2 软件管理
通过apt或yum安装的服务会自动注册到系统服务表中，可以通过systemctl进行管理
```shell
systemctl start ssh #启动ssh服务
systemctl stop ssh #停止ssh服务
systemctl status ssh  #查看ssh服务运行状态
systemctl enable ssh  #设置开机自启ssh
systemctl disable ssh  #关闭开机自启ssh

systemctl list-unit-files --type=service  #列出所有安装的服务
```

#### 4.4 压缩解压
linux中常见两种压缩文件格式：tarball(.tar)和gzip(.gz)，两种压缩方式可以同时使用
tar通常只是简单的归档，文件体积小幅减少。解压命令：
```shell
tar -zcvf dist.tar.gz index.html index.css #将index.html和index.css压缩到dist.tar.gz中，-z代表同时使用gzip压缩，-c表示压缩，-v表示显示压缩进度、-f表示后一个参数为压缩后的文件名
tar -zxvf dist.tar.gz -C dist #将dist.tar.gz解压到dist目录下，-x代表解压，-z代表解压gzip，-f表示要解压的文件，-C表示解压目的目录
```
zip压缩与解压：
```shell
apt install zip -y #安装zip软件包
zip dist.zip index.html index.css #将index.html和index.css压缩到dist.zip中
zip -r dist.zip test #将test目录递归压缩到dist.zip中
unzip dist.zip
```
