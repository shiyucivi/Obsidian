#### 2.1 目录与文件
linux中只有一个根目录/，所有文件都在根目录下。
linux必须选择一个目录作为命令行的工作目录。
==liunx启动后会默认将/home下当前用户的目录作为工作目录==，例如user1用户会进入`home/user1`
```shell
ls #平铺列出当前目录下的所有目录和文件
ls -l #以列表形式列出当前目录下的目录和文件
pwd #当前目录的绝对路径

cd #进入或回到当前用户的工作目录
cd / #进入根目录
cd ~ #进入home目录

mkdir dist #在当前目录下创建一个dist目录
mkdir app/dist/index -p #-p代表如果这个层级的其中一个目录不存在，那么创建

touch index.html #在当前目录下创建一个index.html文件

cat index.html #使用cat查看index.html
more index.html #使用more查看index.html，more中可以通过空格键翻页

cp -r dist distbak #复制dist到当前目录下并命名为distbak，-r表示递归复制
mv dist/index.html distbak #将dist目录下的index.html移动到distbak下
rm -rf distbak #删除distbak，-r表示递归删除，-f表示强制删除
rm -rf dist* #删除名称以dist开头的目录或文件
rm -rf *dist #删除名称以dist结尾的目录或文件
rm -rf *dist* #删除名称中包含dist的目录或文件
```
非root用户默认只有在本用户的目录下创建目录的权限

#### 2.2 查找

which命令用于查找一个命令所在的文件
find可以查找任意类型的文件
```shell
which cd #查找cd命令所在的执行文件

find /home/user1 -name "index" #在/home/user1目录下查找名为index的文件
find / -type d -name "*config*" #在根目录下查找名称中包含 "config" 的目录

whereis nginx #用于查找nginx的二进制文件位置
```

#### 2.3 echo、tail、cut与重定向写入

echo命令用于在命令行中输出参数中的内容，echo的参数最好用""或反引号包裹，echo通常用于在bash脚本中执行以输出指定的内容。
 `>`和`>>`操作符可以将一个命令的执行结果重定向写入到指定文件中
```shell
echo "hello,linux"
hello,linux

echo "hello,linux" > index.html #将字符串hello,linux覆盖写入index.html中
echo "hello,linux" >> index.html #将字符串hello,linux追加写入index.html中

ls -lh >> index.html #将当前目录下的内容以列表格式写入到index.html中
```
tail用于打印文件最末尾的内容，默认输出最后10行的内容
```shell
tail -f index.html #打印index.html最后10行的内容，-f表示持续追踪，文件发生变化时输出内容也更新
```
cut命令可以用于展示一些结构化文本例如CSV等类似列表文本，用于从每行文本中截取需要的字段
```shell
cut -d: -f1 /etc/group #-d选项表示分隔符，:表示每一行文本使用:作为分隔符，-f表示field字段，1表示第一个字段。整个命令的含义是将/etc/group文件中每一行使用:进行隔断，并展示每一行隔断后的第一个字符串
```
#### 3 vi编辑器

vi编辑器通常有三种工作方式：
1 命令模式Command mode
进入编辑文件时默认为命令模式，所按下的按键都被理解为命令并立即执行
2 输入模式Insert mode
按 i 、a、o、I、A、O进入，可以自由编辑文件，按esc退回命令模式
3 底线命令模式Endline mode
按 : 进入，在最下方的底线命令行输入命令并按enter进行执行，通常用于保存退出等

命令模式下常用按键：
/    进入搜索模式，在底线命令行中输入内容并进行搜索
n   搜索下一个
a    搜索上一个
u    撤销修改
p    粘贴
底线命令模式的常用命令：
:wq    保存并退出
:wq!   强制保存并退出，!代表强制操作
:q       直接退出
:q!      强制退出
